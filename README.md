# WA Multi Profile Panel

Tool GUI untuk update foto profil WhatsApp multi akun sekaligus.

## Fitur
- Multi session
- Upload foto massal
- Progress bar realtime
- Status per akun

## Cara Pakai
1. npm install
2. npx playwright install
3. npm start

## Catatan
Gunakan dengan bijak. Ini menggunakan automation (bukan API resmi).
